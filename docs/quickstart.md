# AMR Monitor 快速入门指南

## 1. 基本操作

### 1.1 启动程序

```bash
# 启动ROS核心
roscore

# 启动监控器
roslaunch amr_monitor monitor.launch

```

### 1.2 界面说明

- 左侧：数据可视化区域
  - IMU 数据图表
  - 里程计数据图表
  - 激光雷达数据显示
- 右侧：参数控制区域
  - AMCL 参数调整
  - DWA 参数调整
  - 代价地图参数设置

### 1.3 基本功能

1. 数据监控
   - 实时数据图表显示
   - 鼠标缩放和平移
   - 自动刷新
2. 参数调整
   - 直接输入数值
   - 使用滑块调节
   - 参数实时生效
3. 数据记录
   - 点击"开始记录"
   - 自动保存 CSV 文件
   - 文件包含时间戳

## 2. 高级功能

### 2.1 数据分析

1. 打开分析工具
   - 菜单 -> 工具 -> 数据分析
2. 选择分析方法
   - 基础统计
   - 频谱分析
   - 相关性分析
   - 异常检测
3. 查看结果
   - 数值结果
   - 图形显示
   - 导出报告

### 2.2 自定义配置

1. 话题配置
   - 菜单 -> 配置 -> 话题配置
   - 修改话题名称
   - 验证话题有效性
2. 界面定制
   - 切换主题
   - 调整布局
   - 保存首选项

### 2.3 数据回放

1. 加载数据文件
   - 选择 CSV 文件
   - 自动加载数据
2. 控制回放
   - 播放/暂停
   - 调整速度
   - 跳转位置
